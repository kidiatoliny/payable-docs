This site mirrors the `docs/` tree of [@akira-io/payable](https://github.com/akira-io/payable).
The core knows only contracts, DTOs, actions, value objects, and state machines - never a provider
SDK, HTTP framework, or database client. Money is always handled in minor units through a `Money`
value object, so monetary logic never touches floats.
